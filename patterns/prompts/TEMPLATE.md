# Pattern Title

## Prompt Template

```text
You are helping with {{task_goal}}.
Use {{constraints}} and produce {{output_format}}.
```

## Variables

- `task_goal`: Primary objective the prompt must accomplish.
- `constraints`: Hard limits or required guardrails.
- `output_format`: Expected response shape.

## Example Input

Task goal: summarize recent reliability findings.
Constraints: concise bullets, no speculation.
Output format: issue list with next actions.

## Example Output Shape

- Issue 1: <description>
- Issue 2: <description>
- Next actions:
  - <action>
