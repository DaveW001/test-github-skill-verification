## Root Cause Discovery

The actual canonical OpenCode data directory is `$HOME\.local\share\opencode` (XDG-style). The old `Activate-CompactedDb.ps1` had the correct literal path; the other 5 scripts had the wrong default.

## Fix Applied

All 7 scripts now use `Join-Path $HOME ".local\share\opencode\opencode.db"` as the default. `$HOME` is a built-in PowerShell constant that:
- Resolves to the correct canonical location (confirmed: matches `C:\Users\DaveWitkin\.local\share\opencode`)
- Contains no literal user-name text
- Is cross-platform
- Preserves exact behavioral equivalence

Documentation updated to use generic canonical-location references.
